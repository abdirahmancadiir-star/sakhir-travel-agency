# Product Roadmap

## Current production scope
- Core travel, hotel, tour, visa, cargo, payments, and tracking flows
- Customer dashboard and booking status tracking
- Secure local configuration via .env.example

## Future ideas to keep out of the live app
- Advanced B2B agent portal
- Affiliate and loyalty automation
- AI support, CRM, accounting, and franchise modules
- Additional enterprise reporting and mobile app features

## Next steps
1. Connect real supplier/provider credentials.
2. Add production monitoring, analytics, and audit logging.
3. Review any new modules against the current business scope before exposing them in the app.
